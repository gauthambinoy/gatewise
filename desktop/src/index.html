CA trust

TLS interception needs the GateWise Egress CA trusted on this machine. Fetch it from the gateway if available, then follow the install steps for your OS. 


        

          How to install the root CA
          
Windows: save the PEM as gatewise-ca.crt, then run
            certutil -addstore -user Root gatewise-ca.crt (current user) or open the
            file and choose Install Certificate → Local Machine → Trusted Root.

          
macOS: double-click the PEM to add it to the login keychain,
            then set it to Always Trust; or
            security add-trusted-cert -d -r trustRoot -k ~/Library/Keychains/login.keychain gatewise-ca.pem.

          
Linux: copy to /usr/local/share/ca-certificates/gatewise-ca.crt
            and run sudo update-ca-certificates. Firefox/Chrome keep their own
            stores — import there too if needed.
