# Auvex Console (placeholder)

The React admin console will live here — design tokens + shadcn/ui, auth, the
usage dashboard, the policy editor, and the audit-log explorer. It is built in
**Phase 4** of the roadmap.

It is intentionally **not** a Gradle module: it has its own Node toolchain and
gets wired into the build and CI when Phase 4 begins. This file only exists so
the folder is tracked in git before then.
