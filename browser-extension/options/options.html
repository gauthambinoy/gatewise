GateWise Guard — Options

Configure the GateWise gateway that screens your prompts. Prompt text is sent only to the gateway you set here — nowhere else.

Gateway

The GateWise gateway origin. Prompts are screened via POST /v1/moderations.
Sent as Authorization: Bearer <key>. Stored locally on this device only.

Policy

When off (default, fail-open) a prompt is allowed if the gateway is unreachable, so an outage never stops your work. When on, prompts are blocked whenever the gateway can’t give a verdict — stricter, for regulated environments.
